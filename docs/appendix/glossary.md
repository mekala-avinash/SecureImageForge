# Glossary & References

## Glossary

| Term | Definition |
|---|---|
| **ABAC** | Attribute-Based Access Control |
| **ATO** | Authority to Operate (FedRAMP) |
| **BYOK** | Bring Your Own Key |
| **CAB** | Change Advisory Board |
| **CRD** | Kubernetes Custom Resource Definition |
| **CSI** | Container Storage Interface (used here for Secrets Store CSI) |
| **DSAR** | Data Subject Access Request (GDPR) |
| **DSSE** | Dead Simple Signing Envelope (in-toto/Sigstore) |
| **eBPF** | Extended Berkeley Packet Filter |
| **GitOps** | Declarative ops with Git as source of truth |
| **HITL** | Human-In-The-Loop |
| **IDP** | Internal Developer Platform |
| **IRSA / WIF** | IAM Roles for Service Accounts / Workload Identity Federation |
| **Karpenter** | Open-source K8s node autoscaler/provisioner |
| **mTLS** | mutual TLS |
| **OCI** | Open Container Initiative |
| **PSS** | Pod Security Standards |
| **RPO / RTO** | Recovery Point / Time Objective |
| **SBOM** | Software Bill of Materials |
| **SLSA** | Supply-chain Levels for Software Artifacts |
| **SLO / SLI** | Service Level Objective / Indicator |
| **SoD** | Separation of Duties |
| **SPIFFE / SPIRE** | Secure Production Identity Framework / runtime |
| **SSDF** | Secure Software Development Framework (NIST SP 800-218) |
| **vCluster** | Virtual cluster inside a host K8s cluster |
